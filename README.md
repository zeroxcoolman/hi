## Getting Started

### This is for linux

```bash
curl -R -O http://www.lua.org/ftp/lua-5.4.4.tar.gz
tar zxf lua-5.4.4.tar.gz
cd lua-5.4.4
make all test
```

### You want to first compile. and then you get your code

```bash
lua main.lua  
```

## It will print out
```
Hello, World!
```
